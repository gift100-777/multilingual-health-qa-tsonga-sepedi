# Multilingual Health QA (Xitsonga & Sepedi)

## Overview
This project aims to build a multilingual health question-answering system for Xitsonga and Sepedi using transformer-based language models.

## Goal
Improve access to reliable health information in low-resource African languages and evaluate model reliability in safety-critical healthcare contexts.

## Languages
- Xitsonga
- Sepedi

## Planned Model
- mT5-small

## Objectives
- Build a multilingual health QA dataset
- Fine-tune a multilingual language model
- Evaluate answer quality and safety
- Study hallucinations and translation errors
- Develop a simple prototype for testing

## Current Status
Early-stage development. Project planning, dataset preparation, and model design are underway.

## Competition
This project is inspired by participation in the Zindi Multilingual Health Question Answering in Low-Resource African Languages Challenge.
